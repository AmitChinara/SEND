Write four modular helper functions to build this system:

calculate_total_and_percentage(marks: dict[str, float]) -> tuple[float, float]: Takes a dictionary of subject marks (e.g., {"Math": 80, "Science": 90}) and returns a tuple (total_marks, percentage). Assume each subject is out of 100.

get_grade(percentage: float) -> str: Takes the calculated percentage and returns the appropriate letter grade ("A", "B", "C", or "F").

is_passed(marks: dict[str, float], passing_mark: float = 40.0) -> bool: Checks if the student scored at or above the passing_mark in every individual subject.

generate_report_card(name: str, marks: dict[str, float]) -> dict: Uses the three helper functions above to return a structured report summary.
