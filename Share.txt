1. Find Maximum Number (Easy)

Given an array, return the largest element.

2. Count Vowels in a String (Easy)

Count how many vowels (a, e, i, o, u) are in a string.

3. Factorial of a Number (Easy)

Find factorial of n using recursion or loop.

4. Check Prime Number (Easy)

Check whether a number is prime.

5. Remove Duplicates from Array (Easy-Medium)

Given an array, return a new array without duplicates.

6. Second Largest Element (Easy-Medium)

Find the second largest number in an array.

7. Move Zeros to End (Medium)

Move all zeros in an array to the end while maintaining order of non-zero elements.

Example:
[0,1,0,3,12] → [1,3,12,0,0]

8. Find First Non-Repeating Character (Medium)

Return the first character in a string that does not repeat.

9. Rotate Array by K Steps (Medium)

Rotate an array to the right by k steps.

Example:
[1,2,3,4,5], k=2 → [4,5,1,2,3]

10. Anagram Check (Medium)

Check if two strings are anagrams of each other.

Example:
"listen" & "silent" → true
