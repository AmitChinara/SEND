1. Two Sum (Easy)

Given an array of integers and a target, return indices of two numbers that add up to the target.

Example:
Input: [2,7,11,15], target = 9
Output: [0,1]

2. Reverse a String (Easy)

Reverse a string without using built-in reverse functions.

3. Palindrome Check (Easy)

Check if a given string or number is a palindrome.

Example:
121 → true, 123 → false

4. FizzBuzz (Easy)

Print numbers from 1 to N:

Multiple of 3 → "Fizz"
Multiple of 5 → "Buzz"
Both → "FizzBuzz"
5. Find Missing Number (Medium)

Given an array containing numbers from 1 to N with one missing, find the missing number.

6. Longest Substring Without Repeating Characters (Medium)

Find the length of the longest substring without repeating characters.

Example:
"abcabcbb" → 3 ("abc")

7. Merge Two Sorted Arrays (Medium)

Merge two sorted arrays into one sorted array.

8. Valid Parentheses (Medium)

Check if a string of brackets ()[]{} is valid.

9. Binary Search (Medium)

Implement binary search on a sorted array.

10. LRU Cache (Hard)

Design an LRU (Least Recently Used) cache with:

get(key)
put(key, value)

Time complexity should be O(1).
